Proyecto: Dashboard de Visualización con Gráficas
Ejecución: Directa desde Git Bash

Objetivo
Desarrollar un dashboard interactivo en Python que muestre diferentes tipos de gráficas utilizando Matplotlib y Flet, con el fin de aplicar técnicas de visualización de datos y organización de interfaces gráficas.

Desarrollo
- Se implementaron cuatro funciones para generar gráficas:
- Barras: ventas por producto.
- Líneas: tendencia de rendimiento mensual.
- Dispersión: muestreo aleatorio de sensores.
- Pastel: distribución porcentual de ventas por categoría.
- Se utilizó la librería flet_charts para integrar las gráficas en la interfaz.
- Se organizó el tablero con un GridView, mostrando las gráficas en un diseño ordenado.
- Se añadió un encabezado y divisores para mejorar la presentación.
- La aplicación se configuró con tema claro y se ejecuta directamente desde Git Bash con el comando python archivo.py.

Resultados
- Las gráficas se generaron correctamente y se muestran en un tablero visual.
- El diseño es claro y permite comparar diferentes tipos de datos en una sola interfaz.
- La ejecución desde Git Bash es sencilla y no requiere empaquetado adicional.

Conclusiones
- La práctica permitió reforzar conocimientos sobre:
- Uso de Matplotlib para diferentes tipos de gráficas.
- Integración de visualizaciones en aplicaciones gráficas con Flet.
- Organización de interfaces mediante GridView.
- Ejecución directa de proyectos Python sin necesidad de compilación en APK.
- Este proyecto constituye un ejemplo funcional de cómo transformar datos en visualizaciones claras y accesibles dentro de una aplicación ejecutable en escritorio.
