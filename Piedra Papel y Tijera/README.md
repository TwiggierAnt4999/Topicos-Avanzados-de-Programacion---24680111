Proyecto: Juego “Piedra, Papel o Tijera”
Compilación: APK para dispositivos móviles

Objetivo
Desarrollar una aplicación interactiva en Python utilizando el framework Flet, que permita jugar “Piedra, Papel o Tijera” contra la computadora. El proyecto busca aplicar conceptos de programación orientada a eventos, manejo de interfaces gráficas y despliegue en formato APK para Android.

Desarrollo
- Se utilizó la librería Flet para construir la interfaz gráfica.
- Se definieron las opciones del juego: Piedra, Papel y Tijera.
- Se implementó la función determinar_ganador() para evaluar las jugadas entre el usuario y la computadora.
- Se empleó un PopupMenuButton para que el usuario seleccione su jugada.
- Se creó un contenedor para mostrar la elección de la computadora y otro para mostrar el resultado (Ganaste, Perdiste, Empate).
- Se añadió un botón de Reiniciar, con un cuadro de diálogo de confirmación.
- Se configuró el diseño con tema oscuro, alineaciones centradas y colores de retroalimentación (verde, rojo, gris).

Compilación en APK
El proyecto fue empaquetado y compilado en formato APK, lo que permite su ejecución en dispositivos Android.
Esto demuestra la portabilidad de aplicaciones hechas en Python con Flet hacia entornos móviles, facilitando la distribución y uso práctico del software.
